S T A C K 

LIFO: "Last In, First Out"

OPERATIONS

1. push(): Add an element on top of the stack
2. pop(): Remove the element on top of stack
3. top(): Return the element on top of stack
4. size(): Return size of stack
5. empty(): Return whether or not stack is empty

Q U E U E

FIFO: "First In, First Out"

OPERATIONS

1. push(): Add an element to the end of the queue
2. pop(): Remove the first element of the queue
3. top(): Return the first element of the queue
